# Published Ito Mirror

This directory is a generated, read-only snapshot of Ito coordination state from `ito/internal/changes`.

- `changes/active/` contains active change proposals.
- `changes/archive/` contains archived change proposals.
- `specs/` contains canonical Ito specs.

Do not edit these files directly. Update Ito state through the normal Ito workflow, then regenerate this mirror from the coordination worktree.
