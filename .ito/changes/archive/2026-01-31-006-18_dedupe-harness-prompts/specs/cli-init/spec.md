## ADDED Requirements

### Requirement: Harness instruction wrappers delegate to `ito agent instruction`

`ito init` SHALL install harness-specific prompt/command wrappers whose managed body delegates to `ito agent instruction <artifact>` so the canonical instruction content is generated by the CLI.

#### Scenario: OpenCode command wrapper uses instruction artifacts

- **WHEN** `ito init` configures OpenCode commands
- **THEN** each generated `.opencode/commands/ito-*.md` managed block SHALL instruct the agent to obtain the full workflow instructions by running `ito agent instruction <artifact> --change <change-id>`

#### Scenario: GitHub Copilot prompt file wrapper uses instruction artifacts

- **WHEN** `ito init` configures GitHub Copilot prompt files
- **THEN** each generated `.github/prompts/ito-*.prompt.md` managed block SHALL instruct the agent to obtain the full workflow instructions by running `ito agent instruction <artifact> --change <change-id>`
