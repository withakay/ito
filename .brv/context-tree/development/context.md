# Domain: development

## Purpose
Contains knowledge about release and installer workflows for distributing Ito binaries.

## Scope
Included in this domain:
- Release artifact naming conventions
- Installer download and verification flows
- Platform-specific installation steps
- Release compatibility fallbacks

Excluded from this domain:
- General application runtime behavior
- Internal installer cleanup for obsolete assets

## Ownership
Ito release/installer workflow

## Usage
Use this domain for documenting how release assets are downloaded, validated, and installed on supported platforms.
